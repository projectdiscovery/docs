## Our Tools

ProjectDiscovery maintains popular open source security tools including:

- **[Nuclei](/opensource/nuclei)**: Fast vulnerability scanner with 6,500+ community templates
- **[httpx](/opensource/httpx)**: Multi-purpose HTTP toolkit for security testing
- **[Katana](/opensource/katana)**: Next-generation web crawling framework
- **[Subfinder](/opensource/subfinder)**: Subdomain discovery tool
- **[Naabu](/opensource/naabu)**: Fast port scanner
- And many more security tools in our ecosystem
